# UniSync Design System

**PROJECT CONTEXT:**
- **Product:** UniSync (University Event Management Platform)
- **Type:** B2B/SaaS style web application for administrators, organizers, and students.
- **Aesthetic:** Modern, clean, high-contrast, similar to Vercel, Stripe, and Linear dashboards.

**THEMES:**
Support for both Light and Dark modes.
- **Light Theme:** Neutral light background, white cards, soft shadows, minimal borders.
- **Dark Theme:** Dark slate background, elevated dark cards, high contrast readable typography, subtle glowing borders.

**BRAND COLOR PALETTE:**
- **Primary:** Teal Blue (`#2FA4C6`)
- **Secondary:** Green (`#2DA44E`)
- **Accent:** Orange (`#F59E0B`)
- **Highlight:** Coral (`#F97316`)

**UI COMPONENTS & SHAPES:**
- **Border Radius:** Rounded (`rounded-xl` or `rounded-2xl` for cards, `rounded-lg` for buttons)
- **Shadows/Elevation:** Soft, diffused drop shadows in light mode. Subtle colored glows or high-contrast borders in dark mode.
- **Typography:** Inter or equivalent modern sans-serif. High contrast headings, legible muted body text. 
- **Forms & Inputs:** Clean input fields with subtle focus rings matched to the primary color.
- **Buttons:** Dynamic animations (hover float, click indent). Primary buttons should use the `#2FA4C6` brand color.

**COMMON LAYOUTS:**
- **Public Layout:** Top centered navigation bar (glassmorphism), expansive hero section.
- **Admin/Dashboard Layout:** Left fixed sidebar for navigation, top header for context/actions, main content area for cards and data tables.
